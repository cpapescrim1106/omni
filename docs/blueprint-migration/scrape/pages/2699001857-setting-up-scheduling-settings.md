Overview

The Scheduling Settings setup node allows you to:

 - Set your schedule start/end time by location 
 - Set the time format

Setting up schedule start/end time

 - At Setup > Scheduling > Settings, navigate to Schedule start/end time and click EDIT.

      2. Set the desired location's start and end time, click  UPDATE.

Selecting the schedule time format

 - At Setup > Scheduling > Settings, navigate to Options and select the desired time format from the drop-down menu
 

The 12-hour clock runs from 1am to noon and then from 1pm to midnight.

The 24-hour clock runs from 0:00 (midnight) to 23:59.
