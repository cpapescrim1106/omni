Users with the Delete appointments user privilege can delete appointments from the schedule. Deleted appointments will be excluded from appointment-related reports, including the Cancelled and No Show Appointments Report, and will not trigger the Cancel and no show automated marketing campaign.

Deleting appointments

 - In the Scheduling tab, right-click the desired appointment.
 - Click Delete event. 

 - The Confirm operation dialog box will appear. Click Yes to confirm the operation.

The appointment is removed from the schedule. Deleted appointments are visible on the schedule by using the Status filter to select <Inactive>. See: for more information.

If the appointment is linked to a patient, the deleted event is found in the patient's Journal. If the deleted event is still in the future, it can be recovered from there.

Deleting repeating appointments Deleting repeating appointments

 - In the Scheduling tab, right-click an instance of the repeating appointment.

 - Click Delete event. 

 - In the Confirm operation dialog box, do one of the following: - Use the drop-down menu to select This instance only. Click OK to delete only the one instance of the repeating event. 

 - Use the drop-down menu to select All instances. Click OK  to delete all instances of the repeating event, past and future.

 - Use the drop-down menu to select This and prior instances. Click OK  to delete the selected and prior instances of the repeating event.
 - Use the drop-down menu to select This and future instances. Click OK  to delete the selected and future instances of the repeating event.
