Controls

Group By

Description

Changes the grouping display within the chart's legends. 

 - Appointment Status (Consolidated)

 - Will consolidate appointment statuses so that all past non-cancelled appointments are considered completed.

 - Appointment Status (Detailed)

 - Displays all different types of appointment status, regardless of if they are past

 - Appointment Type

 - 

 - Sales Opportunity

 - Will mark appointments by if they are sales opportunities or not

Appointment Status

Description

Filters for appointment status. Base value will exclude inactive appointment types.

Opportunity Day Range

Description

Set how close a sale must be to be considered a closed opportunity

Appointment Type

Description

Filters for appointment type. This will include inactive appointment types for historical usage

Patient Linked

Description

Filters for appointments with or without patients

Closing Rate Calculations

Description

Filter to only use Sales Opportunity / Aidable Loss appointments in the closing rate calculations. 

This will only effect the closing rate percentage.

Appointment Counts / Calculations

Total Number of Appointments

Description

Number of appointments matching the filters above

Total Number of Aidable Loss Appointments

Description

Number of appointments that are for patients with aidable loss (IE not NR / Normal) AND match the filters above

Total Number of Sales Opportunity Appointments

Description

Number of appointments that are marked as sales opportunities AND match the filters above

Total Number of Aidable Loss Sales Opportunity Appointments

Description
