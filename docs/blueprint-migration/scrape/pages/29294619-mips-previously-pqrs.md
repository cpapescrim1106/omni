MIPS

 - Certain items (those with CPT codes) allow G-codes to be added for Medicare Merit-Based Incentive Payment System (MIPS) recording. Such items are marked with the shield symbol. The insurance company on file for the patient must have the MIPS eligible box checked in order for the shield symbol to become visible for select CPT codes.

 - Right-click on the item

 - A pop-up will appear.  Click in the G code column and select the measure (G-code).

 - Click in any of the columns to add a modifier.

 - Then click Update.
