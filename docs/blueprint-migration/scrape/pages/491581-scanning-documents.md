These instructions apply to Windows users with a TWAIN-compliant scanner. We offer an alternate scanning mechanism for customers using a different OS or a non-TWAIN scanner. Please contact our technical support team to set that up.

Blueprint OMS Accommodates Documents up to 50MB in Size. This allows scanning more pages into a single document or using a higher scanning resolution.

Scanning documents into a patient file

 - In a patient's Documents tab, click Scan.

 - Select the scanner from the drop-down menu and click Ok.

Recommended scanner is the Epson ES-400 II.

 - Set the scanner preferences and complete the scan.

 - When the scan is complete, the Add document dialog box will appear. In it, set the title and document category of the patient's new document. Optionally, add a description, add a document status, or set the document to display in the Audiology tab.

 - Click Save.
