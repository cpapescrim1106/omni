Printing the schedule

 - On the schedule, click the Printer icon.A Print preview window displays a preview of the printable schedule, in the current view. See:  for more information.

 - In the Print preview dialog box, optionally do any of the following:

 - Uncheck the Landscape checkbox (default) to print the schedule in Portrait view.

 - Check the Stretch to page check box to expand the page margins.

 - Click Print.

 - Click Close.

Printing the schedule using the Daily Schedule Report

 - Click on Tools drop-down menu and select Reports.

2. Click on the Daily Schedule report.

3. The Parameters page will then appear. 

a. Choose the report version you wish to view (PDF or XLS)

b. Select the reporting period start and end dates using the calendar icons

c. Select the clinic you wish to include (hold CTRL to select multiple) or select none to include all clinics. 

d. Select the provider you wish to include (hold CTRL to select multiple) or select none to include all providers.

e. Select the event type you wish to include (hold CTRL to select multiple) or select none to include all event types

f. Choose if you want to display patient names on this report

4. Click Ok.

5. The report will generate. Click on the  icon, found in the top left-hand corner of the screen.

For more information on reports, see: Reports.
