This is a listing of the user privileges and their descriptions. User privileges can be restricted/granted to individual Blueprint users. Click on Setup > User Administration > Users > Highlight a name > R-click > Edit Privileges.

Users must have the 'Maintain users and roles' privilege in order to modify another user's privileges.

Basic:

Privilege Name

Description

Create client

Create a new patient

Deactivate client

Mark patients inactive/deceased

Delete added aids

Delete patient hearing aids with status 'device added'

Delete appointments

Permanently delete appointments (and availabilities)

Delete client photo

Delete a photo for a patient

Delete SMS conversations  

Allow user to delete SMS conversations

Delete user photo or signature

Delete a photo for a user or signature

Edit model of active devices

Change the model of hearing aids with status 'active'

Edit model of added devices

Change the model of hearing aids with status 'device added'

Edit patient referral source

Edit referral source history on patients, appointments, and sales

Enter 3p payment

Enter insurance payments

Maintain SMS Messaging settings 

Allow user to configure SMS Messaging settings in the Setup menu

Manage Recalls

Create/edit patient recalls

Marketing

View marketing campaigns

Merge Client

Merge duplicate patients

Order new aid

Create a patient-specific hearing aid order

Receive payment

Receive patient payments/deposits

Return items to manufacturer
