Overview

This functionality enables the seamless transfer of all upcoming appointments from one provider to another, provided both are available at the same location. Appointments cannot be transferred between different locations.

Transferring events is NOT reversible.

How to transfer appointments

noteTo use this feature, you must have the &lsquo;Transfer events&rsquo; privilege.

 - Select Tools > transfer events

 - Today&rsquo;s date will default in the &lsquo;Period start&rsquo;. Click to change and select the &rsquo;period end'. 

 - Select your calendar location, the &lsquo;From' provider & 'To&rsquo; provider. Click next.

 - Review the list of appointments

 - Click 'Finish'```
