Recall statuses

Viewing the list of recall statuses

 - Click the Setup button on the main toolbar.
 - Click Recall statuses. A list of recall statuses will appear in the Recall statuses panel.

Adding new recall statuses

 - Click the Setup button on the main toolbar.

 - Click Recall statuses.
 - Click Create new.
 - In the Create recall status dialog box, enter new recall status. Optionally, check the Is default checkbox to have that recall status appear by default in the Add recall dialog box when users add recalls for patients.
 - Click Create.

Editing recall statuses

 - Click the Setup button on the main toolbar.

 - Click Recall statuses.
 - In the Recall statuses panel, select the desired status, and click EDIT DETAILS.

 - In the Edit recall status dialog box, make adjustments as needed, and click UPDATE.

Deleting recall statuses

 - Click the Setup button on the main toolbar.

 - Click Recall statuses.
 - In the Recall statuses panel, select the desired status and click Edit details.

 - In the Edit recall statuses dialog box, uncheck the Active check box, and click UPDATE.

Recall types

Viewing the list of recall types

 - Click the Setup button on the main toolbar.
 - Click Recall types. A list of recall types will appear in the Recall types panel.

Adding new recall types

 - Click the Setup button on the main toolbar.

 - Click Recall types.
 - Click CREATE NEW.
 - In the Create recall type dialog box, enter new recall type. Optionally, check the Is default checkbox to have that recall type appear by default in the Add recall dialog box when users add recalls for patients.
 - Click CREATE

Editing recall types

 - Click the Setup button on the main toolbar.

 - Click Recall types.
 - In the Recall types panel, select the desired type, click EDIT DETAILS.

 - In the Recall type dialog box, make adjustments as needed, and click UPDATE.

Deleting recall types

 - Click the Setup button on the main toolbar.

 - Click Recall types.
 - In the Recall types panel, click on the desired type, and do one of the following:

 - Double-click. 
 - Click  EDIT DETAILS

 - In the Edit recall type dialog box, uncheck the Active check box, and click UPDATE.```
