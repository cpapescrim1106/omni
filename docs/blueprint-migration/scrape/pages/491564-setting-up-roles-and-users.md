Roles and users overview

Roles can be set up within Blueprint OMS and given specific privileges and reporting access. When a Blueprint OMS user is assigned a role, the user automatically inherits the privileges and reports associated with the role. Privileges and reporting access can also be assigned on an individual-user basis. This section describes the processes of managing roles and users, changing usernames and passwords, adding providers to the schedule, and adjusting NPI numbers and Tax ID numbers for insurance billing purposes.

Users with the Maintain roles and users user privilege will have access to the User administration portion of the Setup menu.
