Navigating the 3rd party receivables menu

Only available to users with the Maintain Insurers privilege

A sale amount must first be allocated to an insurer in order to appear in the 3rd party receivables menu. See: Allocating costs between the patient and insurers for more information 

 - Click the Accounting drop-down menu and select 3rd party receivables.

 - Select the insurer from the drop-down menu to see a list of allocated open invoices.

 - Optionally, select the Location from the drop-down menu to filter the list of invoices by assigned location.

 - Click in the Submitted column to manually record the date of claim submission. If you need to clear the submission date due to an error, right-click and clear the submission date. 

 - Optionally, click in the Notes column to record internal notes regarding the claim.

Selecting the Print button will load the AR Aging Detail (Insurer) report, which shows all the outstanding claims, along with any notes and the submitted date set. This report can be saved as a CSV file for easy sorting and manipulation.

noteOnce the invoice/claim is paid in full, the line item will be removed from this menu.
