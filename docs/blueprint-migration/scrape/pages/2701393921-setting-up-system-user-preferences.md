Overview

At File > Preferences, users with administrator privileges can change system preferences such as GSI report location, AudioScan report location, and Photo upload location.

At File > Preferences, users can change their individual preferences such as Patient arrival notifications, chat input options, and chat notifications.

System preferences

To modify system preferences, navigate to the File drop-down menu > Preferences.

Theme

Users can now choose between the &lsquo;Classic&rsquo; and &lsquo;System&rsquo; themes in Blueprint OMS.

 - Classic: Uses the original Blueprint OMS interface

 - System: Adopts the look and feel of the clinic&rsquo;s operating system (Windows or macOS), providing a more integrated visual experience

Learn more about the GSI Suite integration here.

Learn more about the AudioScan integration here.

Learn more about photo capture on a mobile device here.

User preferences

To modify your user preferences, navigate to the File drop-down menu > Preferences.

Patient arrival notification preferences

If the patient arrival notification is set to Smart, the patient&rsquo;s name will be automatically hidden if any other patient is listed in the in-clinic monitor.

If the patient arrival notification is set to Enabled, notifications will be displayed despite a patient being listed in the in-clinic monitor.

If the patient arrival notification is set to Disabled, notifications will not be displayed.

ENTER key behavior

Users can choose how messages should be sent based on keystrokes.

Chat notification preferences

Users can choose whether or not a notification dialog appears when a new chat message is received. 

A preview of the chat message can be hidden if Smart mode is activated. If set to Smart, the preview will not display if any other patient is listed in the in-clinic monitor.

Users can choose whether or not a sound is played when a new message is received.
