To utilize the mobile app, please configure the Heidi Health integration within Blueprint OMS in advance. (Steps here.) Please note that this method is optional and not the primary intended use of the application.

Mobile Heidi integration workflow:

 - Start a Heidi session from Blueprint OMS. (More here.)

 - Open the Heidi Health mobile app and click Start session.

 - Click stop when appointment is complete and select your note template.

 - Heidi Health will generate the note for you to review.

 - Return to the browser Heidi Health session you started earlier and refresh the page. This will load the recording & note from the mobile app.

 - Click Push note to send to the patient journal tab in Blueprint OMS.

noteWhen the Heidi Health app is closed and subsequently reopened, it will resume at the most recent session.

When the Heidi Health app is closed and subsequently reopened, it will resume at the most recent session.

If the application is open and displaying the sessions list, please navigate to a different menu option and then return to the sessions list to refresh and display the newest created session.

Have questions about Heidi Health? Click here for Heidi Health articles. Contact Heidi Health here.```
