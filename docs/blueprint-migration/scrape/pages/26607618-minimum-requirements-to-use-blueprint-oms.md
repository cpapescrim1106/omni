Operating systems

 - Windows: Windows 10 or newer

 - Mac: Blueprint OMS only supports Mac-supported OS versions. Please follow this link to stay updated on which versions are supported.

 - Linux

For use on tablets, Windows 8.x Pro or Windows 10 are required (the application will not run on tablets running the Windows 8 (Core) operating system).

Required software

 - 32-bit Java Runtime Environment version 8.x

Native Windows Installer

If using the native Windows installer, it is not necessary to install Java.

  The following functionality is not available on non-Windows platforms:

 - Signature pad integration

 - Quickbooks integration

Display

Minimum display resolution of at least 1280x1024 pixels

System Memory (RAM)

At least 4GB of free RAM after operating system has booted, and all other required services/software have been loaded

Recommended Network Speed

We recommend a broadband internet connection. To test the speed of your internet, you can visit www.speedtest.net to check your download/upload speed. Our recommended download/upload speed depends on the number of computers in the office:

 - For a small office (1-3 computers), we recommend 2 Mbps download and .5 Mbps upload.  

 - For larger offices (3+ computers), we recommend at least 5 Mbps download and 2 Mbps upload.
