Downloading instructions

 - Go to Settings.

   

 - For iOS 18 and newer, scroll to the bottom of Settings and go to Apps, then Calendar.

For iOS 14-17, go to Calendar directly from Settings.

For iOS versions older than 14, go to Accounts & Passwords.

 - Go to Accounts (skip this step for iOS 13 or older). 

 - Go to Add Account.

 - Go to Other.

 - Go to Add Subscribed Calendar.

 - Enter the iCal URL retrieved from Blueprint and select Next. You can find more information on the iCal URL here. 

 - Edit the description to change the calendar name and select Save.

 - Go to your calendar and select the calendar(s) you would like to view
