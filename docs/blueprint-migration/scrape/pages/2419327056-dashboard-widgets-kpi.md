```text
When the dashboard is first displayed, the default Date range is This month.

This can be changed using the Date range button above the dashboard widgets.

Financial

Net revenue

Description

Displays total Net revenue (Sales - Returns) for the date range.

Gross profit

Description

Displays Gross profit (Net revenue - Net cost) for the date range.

Net HA revenue

Description

Displays Net HA revenue (HA sales - HA returns) for the date range.

Note: For US clinics, units covered by managed care companies are excluded from this calculation.

Net HA cost

Description

Displays Net HA cost (Cost of HA sales - Cost of HA returns) for the date range.

Note: For US clinics, units covered by managed care companies are excluded from this calculation.

Order backlog

Description

Displays potential revenue from all undelivered orders (new orders as well as repairs)

Unit sales

Gross units fit

Description

Displays the total number of hearing aids fit during the date range

Note: For US clinics, units covered by managed care companies are excluded from this calculation.

Net units fit

Description

Displays the net hearing aids fit (total units fit - units returned - units exchanged) during the date range

Note: For US clinics, units covered by managed care companies are excluded from this calculation.

Returned units

Description

Displays the total number of hearing aids returned during the date range

Note: For US clinics, units covered by managed care companies are excluded from this calculation.

Exchanged units

Description

Displays the total number of hearing aids exchanged during the date range

Note: For US clinics, units covered by managed care companies are excluded from this calculation.

Return rate

Description

Displays the return rate (units returned / total units fit) during the date range

```
