When creating an appointment for a new patient, common usage would be to select "None" when prompted for the patient and to simply type in the patient&rsquo;s name for the appointment description. Creating a full patient file would be deferred until the patient&rsquo;s first visit. When scheduling an appointment for a new patient, it&rsquo;s now possible to link the appointment to a new QuickAdd record which captures basic patient information.

The advantages of using a QuickAdd are:

 - The appointment is automatically linked to the QuickAdd file (which will later become the full patient file)

 - The patient&rsquo;s phone number and referrer can be captured (and will later automatically become part of the full patient file)

 - The patient can receive an automated appointment reminder (if a mobile phone number or email address was entered)

 - Other useful information such as an alternate contact, or notes regarding what was discussed can optionally be captured

Other QuickAdd-related features are:

Viewing, filtering, and creating QuickAdds from the Patient browser

 - QuickAdds are highlighted in the Patient browser with an icon beside their names.

 - QuickAdds are included in the list by default but can be hidden by clicking the Show QuickAdds toggle button.

 - The QuickAdds only toggle button will hide full patient files, and display only QuickAdd files.

 - The QuickAdd button at the bottom of the Patient browser can be used to create a new QuickAdd file.

Online appointment booking

 - When an appointment is booked online for a person whose information cannot be matched to an existing patient file, a QuickAdd will be created for that person using the data they entered at portal.

Entering QuickAdd Patients

 - Open the Create QuickAdd dialog box from one of two places: Patient browser > QuickAdd, or, alternatively, Appointment > QuickAdd.

 - Enter all required information.

 - Optionally add the following information:

 - Title

 - Gender

 - Middle Initial

 - Patient Grouping

 - Email Address

 - Date of Birth

 - Notes

Some fields on the Create QuickAdd screen are mandatory, indicated by an asterisk (*). Other fields must conform to a specific format (e.g. email address). Fields which are missing mandatory information, or which contain information in an incorrect format, are highlighted in blue.

Opening QuickAdd Files

Open a QuickAdd file by doing one of the following:

1. Double click the patient's name 

2. Highlight the patient and click the  icon located at the bottom of the Patient browser.

Once open, the screen will look similar to a full patient's file with the exception of the Hearing aids, Audiology, Insurers, and Sales history tabs. Once the QuickAdd is converted to a full patient, those tabs will become available.

Converting QuickAdds to full patient files

The QuickAdd can easily be converted to a full patient file later &mdash; typically when the patient is present in the clinic, and it&rsquo;s easier to collect details such as address and insurance information. There are two ways to convert QuickAdds to full patients: manually, and using online forms.

Manually converting a QuickAdd to a full patient

1. Open the patient's file by following the instructions given above

2. Navigate to the Details tab and click Convert to patient towards the bottom of the screen.

3. Enter all required information.

4. Click Create patient.

Some fields on the Create patient screen are mandatory, indicated by an asterisk (*). Other fields must conform to a specific format (e.g. birth date, telephone numbers, and postal codes). Fields which are missing mandatory information, or which contain information in an incorrect format, are highlighted in blue.
