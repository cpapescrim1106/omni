Notifications

To indicate an incoming message, the bubble will flash in green five times and then remain a solid color. Click on thebubble to open the dialog box.

Any unread messages will appear with green text until viewed. To view an unread message, click on the green text. 

Conversation List

If multiple conversations are in progress, there will be a list in the conversation dialog box. Click on any conversation to respond or to view the messages.

Chat conversations are stored only temporarily. They are retained if the user logs off the Blueprint OMS application, but are deleted if the application is closed completely.
