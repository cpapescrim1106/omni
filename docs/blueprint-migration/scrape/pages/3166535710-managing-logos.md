Create logo

For clinics with multiple locations, unique logos can be added for each location.

 - Click the Setup button on the main toolbar

 - Click Locations and Logos

 - Select Create new

 - Click on the image icon and select the desired file

 - Give the logo a name and select which locations the logo should be assigned to

 - Check the Select all checkbox to apply the new logo to all locations

 - Click Create

Edit logo

 - Click the Setup button on the main toolbar

 - Click Locations and Logos

 - In the Logos panel, click on the desired location, and do one of the following:

 - Double-click.

 - Click EDIT DETAILS

 - Make the desired changes and select Update
