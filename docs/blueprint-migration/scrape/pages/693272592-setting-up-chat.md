Setting up Chat

This is only visible to users with an Owner role under User administration.

 - Click Setup on the main toolbar.
 - Click Chat.
 - Click Enable Chat.

When chat is enabled, the enter key can be set to work in one of two ways within chat. You can decide which one suits your clinic the best.

 - Press ENTER to send a message and Shift + ENTER to create a new line.
 - Press ENTER to start a new line and Shift + ENTER to send a message.

Enable chat will enable chat for all users and locations. Chat cannot currently be enabled on a per-location basis, but it can be disabled on a per-user basis.

By default, every user has the View chat privilege. To disable chat for a user, remove their View chat privilege by navigating to Setup > User administration > Users, right-click on a user, select Edit privileges, uncheck the View chat privilege under Basic, and finally click Update.
