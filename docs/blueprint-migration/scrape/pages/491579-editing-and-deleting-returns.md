Editing returns

If a refund is applied against a return, you will need to remove the refund before you can delete the return. - On the Patient browser tab, open the patient&rsquo;s file.
 - Click the Sales history tab.
 - Right-click the desired return and select Edit return details.
 - Adjust the return as necessary, and click Update return.
QuickBooksAn update to a return will not transfer to QuickBooks. First, delete the return and then redo the return.

Editing return application(s) Editing return application(s)

 - On the Patient browser tab, open the patient&rsquo;s file.
 - Click the Sales history tab.
 - Right-click the desired return and select Edit return application(s).
 - To unapply return credits, in the Edit return applications dialog box, un-check the box to the left of the TXN ID number, and click Save.
 - To adjust the applied amount, in the Edit return applications dialog box, double-click in the Amount applied field and enter the desired amount, click out of the cell or click Enter on the keyboard to save the change, and click Save.
